# Smart Farmer Assistant

Modern frontend for an AI-powered Smart Farmer Assistant built with React, Vite, Tailwind CSS, React Router, Framer Motion, React Hook Form, Recharts, Axios, and Context API.

## Features

- Responsive auth flow with login, register, forgot password, and OTP verification
- Protected dashboard layout with sidebar, navbar, dark mode, and language switcher
- AI Crop Doctor, AI Chat, Weather, Market Prices, Fertilizer, Expenses, Profit, Journal, Schemes, Community, Reports, and Settings pages
- Reusable component system and mocked data for future API integration
- Lazy-loaded routes and production build-ready structure

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```
